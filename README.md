# Taskease - Student Task Manager

**Taskease** is a Streamlit-based task management app designed specifically for students. It helps students organize, track, and prioritize their tasks, projects, and deadlines efficiently.

---

## **Features**
- ✅ User Login & Sign Up with secure password hashing  
- ✅ Add tasks with **title, description, category, and deadline**  
- ✅ Task status management: Pending / Completed  
- ✅ **Premium dashboard** with card-style tasks and color-coded notifications  
  - Red for Overdue tasks  
  - Yellow for tasks Due Today  
- ✅ Expandable task descriptions for clarity  
- ✅ **Progress bars**: overall and category-wise  
- ✅ **Sidebar summary**: Upcoming tasks  
- ✅ **Search & Filter** tasks by category and status  
- ✅ Analytics page with **metrics and category progress**

---

## **Tech Stack**
- Python 3  
- Streamlit  
- SQLite for database  
- bcrypt for password hashing
